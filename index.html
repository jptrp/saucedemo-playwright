Quick Start Guide

1

Clone the Repository

git clone https://github.com/jptrp/saucedemo-playwright.git
cd saucedemo-playwright
2

Install Dependencies

npm install
npx playwright install
3

Run Tests

# Run all tests
npx playwright test

# Run in headed mode
npx playwright test --headed

# View HTML report
npx playwright show-report
